
# Summary of this project

This project is a complex backend project that is built with nodejs, expressjs, mongodb, mongoose, jwt, bcrypt, and many more. This project is a complete backend project that has all the features that a backend project should have.
We are building a complete video hosting website similar to youtube with all the features like login, signup, upload video, like, dislike, comment, reply, subscribe, unsubscribe, and many more.

Project uses all standard practices like JWT, bcrypt, access tokens, refresh Tokens and many more. We have spent a lot of time in building this project and we are sure that you will learn a lot from this project.

Video Hosting Platform Backend

Tech Stack: Node.js, Express.js, MongoDB, JWT, JavaScript, Multer, Cloudinary, Postman

Key Features:

Developed REST APIs for user authentication and profile management using JWT.
Implemented video upload functionality using Multer and Cloudinary.
Built features for video publishing, updating, and deletion.
Created APIs for likes, comments, subscriptions, and playlists.
Implemented watch history and user channel management.
Added secure authorization for protected routes.
Designed MongoDB schemas and aggregation pipelines for efficient data retrieval.
Tested and documented APIs using Postman.
Followed MVC architecture and reusable middleware patterns.
